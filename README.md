# Kids Math 🧮

A fun, interactive web application designed to help children practice and improve their basic mathematical skills. 

### 🚀 Live Demo
**Play the game here:** [Kids Math - Live Deployment](https://jaswinderwadali.github.io/kids_math/)

---

## 🌟 Features
* **Interactive Learning:** Engaging interface to keep kids interested while learning.
* **Basic Arithmetic:** Covers fundamental math operations like Addition, Subtraction, Multiplication, and Division.
* **Instant Feedback:** Helps children learn from their mistakes immediately.
* **Accessible Anywhere:** Fully web-based and responsive, playable on desktop, tablet, or mobile.

---

## 💻 Tech Stack
* **HTML5:** For the structure of the web application.
* **CSS3:** For styling and creating a kid-friendly, colorful user interface.
* **JavaScript:** For the game logic, math problem generation, and interactive elements.
* **GitHub Pages:** For hosting and deployment.
